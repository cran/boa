boa.pardesc {boa}R Documentation

Global Parameters Descriptions

Description

Returns descriptive information on the global parameters.

Usage

boa.pardesc()

Value

A character matrix whose rows and columns ("group", "method", "desc", "par", "note") contain the global parameters and the corresponding descriptors (group = "Analysis", "Data", or "Plot"; method = subgroup classification; desc = parameter description; par = parameter name name, note = information concerning the possible values for the parameter).
